Check if your package was installed globally

Can be useful if your CLI needs different behavior when installed globally and locally.

Install

$ npm install is-installed-globally

Usage

``‘js const isInstalledGlobally = require('is-installed-globally’);

// With npm install your-package console.log(isInstalledGlobally); //=> false

// With npm install --global your-package console.log(isInstalledGlobally); //=> true ```
